# SocialConnectApp 📱

A professional, feature-rich social media application built with **React Native** and **Firebase**. This project demonstrates modern mobile app development practices, including real-time database management, authentication, and a polished user interface.

## ✨ Features

- **Authentication**: Secure Login and Signup using Firebase Auth.
- **Dynamic Home Feed**: View posts from users with real-time updates.
- **Post Management**: Create, edit, and delete text-based or image posts.
- **User Profiles**: Custom profiles with bio, profile picture updates, and activity tracking.
- **Social Graph**: Follow/Unfollow system with dedicated Followers and Following lists.
- **Real-time Messaging**: Instant chat system with a dedicated Chat List and messaging UI.
- **Dark Mode Support**: Seamlessly toggle between Light and Dark themes via Settings.
- **Search**: Discover other users within the platform.

## 🛠 Tech Stack

- **Frontend**: React Native (JavaScript/TypeScript)
- **Navigation**: React Navigation (Stack & Tabs)
- **Backend**: Firebase (Firestore, Authentication, Storage)
- **Icons**: React Native Vector Icons (Ionicons)
- **Image Handling**: React Native Image Picker
