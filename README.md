# SocialConnectApp 📱
A professional social media app built with **React Native** and **Firebase**.

## ✨ Features
* **Authentication:** Secure Login & Signup via Firebase.
* **Posts:** Create text and image posts.
* **Interactions:** Like and Comment system on every post.
* **Profiles:** Personalized user profiles with bio and picture.
* **Search:** Find other users in real-time.

## 🛠️ How to Run
1. `npm install`
2. `npx react-native run-android`
